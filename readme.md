# GitBranch Far Manager plugin

Plugin helps Far Manager to show git branch at command prompt. It setups `%SH_GIT%`  and `%SH_DIR%` to git info for repository and directory on active panel. You should customize "Command line settings" to insert for example `(b)%SH_DIR%(d)%SH_GIT%(a)‚ùØ` in command line prompt format string. Branch name will be shown at command prompt:
